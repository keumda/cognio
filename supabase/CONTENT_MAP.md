## 1. 결과 유형 (10가지)

`src/data/resultTypes.ts`

5단계 점수 중 가장 낮은 단계(들)를 기반으로 유형이 결정됨.

### 단일 유형 (5개)

| 키 | 이모지 | 이름 | 설명 요약 | 시적 문구 |
|---|---|---|---|---|
| `1` | 🧱 | 보이지 않는 벽 뒤의 사람 | 신뢰 미형성, 혼자 해결해야 한다는 생존 전략 | "혼자여도 괜찮다고 말하는 그 뒤에, 누군가를 기다리는 마음이 있습니다." |
| `2` | 🛡️ | 감정의 갑옷을 입은 사람 | 수치심이 선택을 지배, 완벽하지 않으면 숨김 | "감추고 싶은 그 모습 뒤에, 진짜 당신이 숨 쉬고 있어요." |
| `3` | ⏸️ | 출발선에 선 채 멈춘 사람 | 실패 공포로 시작 불가, 게으름이 아닌 공포 | "출발하지 못한 오늘이 있어도, 당신의 가능성은 줄지 않았어요." |
| `4` | 🏋️ | 증명의 무게를 진 사람 | 끊임없는 성과 압박, 쉬면 뒤처지는 불안 | "끊임없이 증명하는 무게 아래, 진짜 당신은 이미 충분합니다." |
| `5` | 🔍 | 나를 찾고 있는 사람 | 정체성 혼란, 진짜 나를 모르는 느낌 | "길을 잃은 것처럼 느껴져도, 찾고 있다는 것 자체가 답입니다." |

### 콤보 유형 (5개)

두 번째로 낮은 단계도 50% 이하이고, 해당 조합이 존재할 때:

| 키 | 이모지 | 이름 | 설명 요약 | 시적 문구 |
|---|---|---|---|---|
| `1,2` | 🐚 | 세상에서 가장 작은 갑옷 | 타인도 자신도 믿기 어려운 상태 | "작은 갑옷을 벗어도 괜찮은 세상이 있다는 걸, 조금씩 알아가봐요." |
| `1,3` | 🌱 | 안전한 시작을 찾는 사람 | 실패하면 도와줄 사람이 없을 것 같은 공포 | "안전한 첫 걸음은 생각보다 작아도 돼요." |
| `2,3` | 😶 | 시작이 두려운 완벽주의자 | 수치심이 시작을 막는 패턴 | "부끄러움 없이 시작하는 법을, 함께 찾아갈 수 있어요." |
| `2,4` | 🔒 | 완벽이라는 감옥에 갇힌 사람 | 완벽하지 않으면 가치 없다는 믿음 | "완벽하지 않아도 사랑받을 수 있다는 것을, 다시 배울 수 있어요." |
| `3,4` | 💫 | 달려야 하는데 출발할 수 없는 사람 | 성취 압박 + 시작 불능의 내적 갈등 | "달려야 한다는 압박을 잠시 내려놓아도, 당신의 자리는 사라지지 않아요." |

### 유형 결정 로직

1. 5단계 퍼센티지 계산
2. 가장 낮은 단계 = `lowest`, 두 번째 = `secondLowest`
3. `secondLowest`도 50% 이하이고, 콤보 조합 `["1,2", "1,3", "2,3", "2,4", "3,4"]`에 해당 → 콤보
4. 아니면 → 단일 (`lowest` 단계 번호)

## 2. 연결 인사이트 (12개)

`src/lib/connectionInsights.ts` — ConnectionInsight 컴포넌트

### 코스 추천 통합 로직

**앱 전체에서 코스 추천이 일관되도록 pathway 코스를 항상 우선합니다.**

실제 표시되는 2개 코스 = `[pathway 코스, 인사이트 패턴 코스]`

1. 유저의 pathway에 매칭되는 코스를 항상 1번째로 포함
2. 인사이트 데이터의 courseIds 중 pathway 코스를 제외한 첫 번째를 2번째로
3. 결과적으로 pathway 코스가 ConnectionInsight, RecommendationCards(Hero) 모두에서 동일하게 1순위

예: pathway=`attachment`, 연결 패턴=`2,3` (수치심→시작불능)
- 인사이트 데이터: `["perfectionism", "initiative"]`
- 실제 표시: **관계 패턴 리셋** (pathway) + **완벽주의 해제** (인사이트 1번째)

### 콤보 인사이트 (7개) — 두 단계가 50% 이하일 때

UI: [단계 카드] → [단계 카드]

| 키 | 제목 | 인사이트 요약 | 기본 코스 (pathway로 1번째가 대체됨) |
|---|---|---|---|
| `1,2` | 신뢰 부족 → 수치심 강화 | 신뢰 약하면 자율 시도 시 수치심 악순환 | 관계 패턴 + 완벽주의 |
| `1,3` | 신뢰 부족 → 안전한 시작 불가 | 실패하면 아무도 받아주지 않을 것 같은 복합 공포 | 관계 패턴 + 시작의 기술 |
| `1,5` | 신뢰 부족 → 정체성 불안정 | 안전한 연결 없이 자기 탐색 어려움 | 관계 패턴 + 나를 찾는 여정 |
| `2,3` | 수치심 → 시작 불능 | 못하면 부끄러워서 시작 못함, 미루기의 뿌리 | 완벽주의 + 시작의 기술 |
| `2,4` | 수치심 + 열등감 = 완벽주의 감옥 | 완벽하지 않으면 가치 없다는 패턴, 번아웃 원인 | 완벽주의 + 번아웃 |
| `3,4` | 시작 불능 + 성취 압박 = 만성 스트레스 | 해야 하는데 못 하겠다는 자기 비난 루프 | 시작의 기술 + 번아웃 |
| `4,5` | 성취 의존 → 정체성 공허 | 성과 빼면 나는 누구인가, 번아웃 시 극심한 공허 | 나를 찾는 여정 + 번아웃 |

### 단일 인사이트 (5개) — 한 단계만 50% 이하일 때

UI: [단계 카드] → [파급 영역 점선 박스]

| 키 | 제목 | 인사이트 요약 | 기본 코스 (pathway로 1번째가 대체됨) |
|---|---|---|---|
| `1` | 신뢰 부족 → 관계와 자율성 영향 | 기대기 어려움, 혼자 해결해야 하는 부담 | 관계 패턴 + 완벽주의 |
| `2` | 수치심 → 행동과 관계 영향 | 실수 공포가 시도를 막고, 가면을 쓰게 함 | 완벽주의 + 시작의 기술 |
| `3` | 시작 어려움 → 성취와 자신감 영향 | 시작 못함 → 성취 없음 → 자신감 하락 악순환 | 시작의 기술 + 번아웃 |
| `4` | 성취 압박 → 에너지와 정체성 영향 | 더 잘해야 한다는 끊임없는 압박, 만성 번아웃 | 번아웃 + 나를 찾는 여정 |
| `5` | 정체성 혼란 → 선택과 관계 영향 | 타인 기대에 맞추는 삶, 반복적 전환 | 나를 찾는 여정 + 관계 패턴 |

### 인사이트 선택 로직

1. 50% 이하 단계가 2개 이상 → 콤보 키 시도
2. 콤보 키 없거나 1개뿐 → 단일 인사이트
3. 0개 → 미표시

## 3. 심화 분석 연결 문구

| `avoidance` | 🫥 회피형 완벽주의 | "시작 자체를 피하는 패턴이 더 강화되어, 해야 할 일을 계속 미루게 될 수 있어요." |
| `proving` | 📊 증명형 완벽주의 | "끊임없이 증명해야 한다는 압박이 번아웃을 가속시킬 수 있어요." |

| `dismissive` | 🔵 회피형 | "도움을 요청하지 않는 패턴이 혼자 감당하려는 부담을 키울 수 있어요." |

| `secure` | 💚 안정형 | "안정적 관계 경험이 다른 단계의 회복에도 좋은 토대가 됩니다." |

| `acceptance` | 🤲 감정 수용 | "감정을 억누르려는 시도가 오히려 더 큰 스트레스로 돌아올 수 있어요." |
| `strategy` | 🧰 감정 전략 | "감정은 알지만 대처 방법을 모르는 무력감이 반복될 수 있어요." |

+ 인식/수용/전략 점수 표시

### initiation (주도성) — primaryChallenge별 연결 문구

| primaryChallenge | 유형명 | 연결 문구 |

| `energy` | 🔋 에너지 고갈형 | "에너지 고갈로 변화를 시도할 여력 자체가 남아있지 않을 수 있어요." |
| `focus` | 🌀 방향 상실형 | "방향을 잡지 못해 시작과 포기를 반복하는 패턴이 강화될 수 있어요." |

+ 실패공포/에너지/집중 점수 표시
