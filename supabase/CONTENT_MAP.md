## 2. 연결 인사이트 (12개)

`src/lib/connectionInsights.ts`

### 콤보 인사이트 (7개) — 두 단계가 50% 이하일 때

두 약한 단계가 서로 영향을 주는 패턴. UI: [단계 카드] → [단계 카드]

| 키 | 제목 | 추천 코스 2개 |
|---|---|---|
| `1,2` | 신뢰 부족 → 수치심 강화 | 관계 패턴 리셋 + 완벽주의 해제 |
| `1,3` | 신뢰 부족 → 안전한 시작 불가 | 관계 패턴 리셋 + 시작의 기술 |
| `1,5` | 신뢰 부족 → 정체성 불안정 | 관계 패턴 리셋 + 나를 찾는 여정 |
| `2,3` | 수치심 → 시작 불능 | 완벽주의 해제 + 시작의 기술 |
| `2,4` | 수치심 + 열등감 = 완벽주의의 감옥 | 완벽주의 해제 + 번아웃 리커버리 |
| `3,4` | 시작 불능 + 성취 압박 = 만성 스트레스 | 시작의 기술 + 번아웃 리커버리 |
| `4,5` | 성취 의존 → 정체성 공허 | 나를 찾는 여정 + 번아웃 리커버리 |

### 단일 인사이트 (5개) — 한 단계만 50% 이하일 때

하나의 약한 단계가 다른 영역으로 파급되는 효과. UI: [단계 카드] → [파급 영역 점선 박스]

| 키 | 제목 | 추천 코스 2개 |
|---|---|---|
| `1` | 신뢰 부족이 관계와 자율성에 미치는 영향 | 관계 패턴 리셋 + 완벽주의 해제 |
| `2` | 수치심이 행동과 관계에 미치는 영향 | 완벽주의 해제 + 시작의 기술 |
| `3` | 시작의 어려움이 성취와 자신감에 미치는 영향 | 시작의 기술 + 번아웃 리커버리 |
| `4` | 성취 압박이 에너지와 정체성에 미치는 영향 | 번아웃 리커버리 + 나를 찾는 여정 |
| `5` | 정체성 혼란이 선택과 관계에 미치는 영향 | 나를 찾는 여정 + 관계 패턴 리셋 |

### 인사이트 선택 로직

1. 50% 이하 단계가 2개 이상 → 가장 낮은 2개로 콤보 키 생성
2. 콤보 키가 `connectionInsights`에 있으면 → 콤보 인사이트 표시
3. 없거나 50% 이하가 1개뿐이면 → 가장 낮은 단계로 `singleStageInsights` 표시
4. 50% 이하 단계가 0개 → 인사이트 미표시

## 4. 심화 분석 (pathway별)

`src/lib/additionalScoring.ts`

### perfectionism (완벽주의)
- 3가지 유형: `shame`(수치심형), `avoidance`(회피형), `proving`(증명형)
- 각 2문항씩 6문항 → primaryType 결정

### attachment (애착)
- 2축: `anxiety`(불안), `avoidanceAxis`(회피)
- 중앙값(9) 기준 4분면: `secure`, `anxious`, `dismissive`, `fearful`

### emotion (감정조절)
- 3영역: `awareness`(인식), `acceptance`(수용), `strategy`(전략)
- weakestArea = 가장 낮은 영역

### initiation (주도성)
- 3영역: `fear`(실패 공포), `energy`(에너지), `focus`(집중)
- primaryChallenge = 가장 높은 어려움 (높을수록 문제)
